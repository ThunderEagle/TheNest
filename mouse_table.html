Mouse Table

 

This was one of my first projects.  As you can see, the keyboard tray for my computer desk doesn't allow for enough room for a mouse pad right next to it.  I put up with this for a long while, but when playing many games, you really need that mouse right next to the keyboard.  I have done several things to do this in the past.  One was to put part of my keyboard on my server case and the mouse would fit on the drawer.  The second was to open the file cabinet drawer you see there behind my table and put an empty box on top of the hanging folders in there.

 


This is just a simple little table made out of 3/4" MDF.  I dadoed the lower shelf in, and added edge banding all around.  I could have probably given it a much more "finished" look with a router, but this was made before I picked my router up.
